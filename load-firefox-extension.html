Loading Firefox Extension

Note: If you're seeing the error "background.service_worker is currently disabled. Add background.scripts", you need to make sure you're selecting the correct manifest file.

Method 1: Use the batch files

  1. Run prepare-firefox-extension.bat to temporarily rename the manifest files
  2. Load the extension in Firefox (see steps below)
  3. After loading, run restore-manifests.bat to restore the original filenames

Method 2: Manual selection

  1. Open Firefox and go to about:debugging
  2. Click "This Firefox" in the left sidebar
  3. Click "Load Temporary Add-on"
  4. Navigate to the extension directory
  5. IMPORTANT: Select the "manifest-firefox.json" file, NOT "manifest.json"

Verifying the extension loaded correctly

  1. After loading, check if the extension appears in the list under "This Firefox"
  2. Look for the extension icon in the Firefox toolbar
  3. If you don't see the icon, try restarting Firefox
  4. Check the browser console (F12) for any errors
